# 全球湖泊旱涝急转事件统计分析报告

**生成日期**: 2026-04-18

**数据来源**: lakecentroid 数据库 `monthly_transition_abrupt_transitions` 表

---

## 1. 方法概述

### 1.1 旱涝急转定义

本分析基于 **月距平分位数阈值法** 识别旱涝急转事件：

1. 对每个湖泊计算多年月均气候态（每月均值）
2. 计算各月的距平值 = 当月水面积 - 气候态
3. 使用 10% 和 90% 分位数作为极端事件阈值：
   - **极端干旱 (extreme_low)**: 距平值 ≤ 10%分位数
   - **极端洪涝 (extreme_high)**: 距平值 ≥ 90%分位数
4. 检测相邻月份的极端状态跳变：
   - **旱→涝 (low_to_high)**: 从极端干旱跳变到极端洪涝
   - **涝→旱 (high_to_low)**: 从极端洪涝跳变到极端干旱

### 1.2 分析范围

- **时间跨度**: 2001年 - 2023年（共23年）
- **数据记录**: 共 1,853,952 条旱涝急转事件

## 4. 主要发现

1. **事件普遍性**: 在分析的时间范围内，全球约 **62.4万个湖泊** 至少发生过一次旱涝急转事件

2. **频率适中**: 平均每湖约 **3次** 事件，中位数仅 **2次**，说明整体频率相对较低

3. **类型均衡**: 旱→涝和涝→旱两种类型事件数量几乎相等，反映水循环的对称性

4. **少数极端**: 极少数湖泊（~2000个）经历了20次以上的事件，这些湖泊值得重点关注，可能具有特殊的生态或气候敏感性

5. **年代趋势**: 2019年以来旱涝急转事件呈上升趋势，可能与气候变化加剧有关

## 5. 附录：SQL 查询

### 年度统计查询

```sql
SELECT
    to_year AS year,
    COUNT(*) FILTER (WHERE transition_type = 'low_to_high') AS drought_to_flood,
    COUNT(*) FILTER (WHERE transition_type = 'high_to_low') AS flood_to_drought,
    COUNT(*) AS total_transitions,
    COUNT(DISTINCT hylak_id) AS lakes_affected
FROM monthly_transition_abrupt_transitions
GROUP BY to_year
ORDER BY to_year;
```

### 湖泊统计查询

```sql
WITH lake_transition_counts AS (
    SELECT
        hylak_id,
        COUNT(*) AS total_transitions,
        COUNT(*) FILTER (WHERE transition_type = 'low_to_high') AS drought_to_flood,
        COUNT(*) FILTER (WHERE transition_type = 'high_to_low') AS flood_to_drought
    FROM monthly_transition_abrupt_transitions
    GROUP BY hylak_id
)
SELECT
    COUNT(*) AS total_lakes_with_transitions,
    ROUND(AVG(total_transitions), 2) AS avg_transitions_per_lake,
    MIN(total_transitions) AS min_transitions,
    PERCENTILE_CONT(0.50) WITHIN GROUP (ORDER BY total_transitions) AS median_transitions,
    MAX(total_transitions) AS max_transitions
FROM lake_transition_counts;
```

## 1. 方法概述

### 1.1 Hawkes过程简介

Hawkes过程（自激点过程）是一种用于建模具有"传染效应"事件的统计方法。在湖泊水面积变化分析中，Hawkes过程能够捕捉以下效应：

- **基率 (μ)**: 极端事件自发发生的背景概率
- **传染系数 (α)**: 一个事件触发后续同类事件的强度
- **衰减率 (β)**: 传染效应的衰减速度
- **谱半径 (ρ = α/β)**: 衡量整体传染强度的稳定性指标

### 1.2 模型结构

本分析使用双状态Hawkes过程，状态定义为：

- **干态 (D)**: 湖泊水面积处于极端干旱状态（低于10%分位数阈值）
- **湿态 (W)**: 湖泊水面积处于极端洪涝状态（高于90%分位数阈值）

模型包含4组参数：
- μ_D, μ_W: 干态和湿态的基率
- α_DD, α_DW, α_WD, α_WW: 各状态间的传染系数
- β_DD, β_DW, β_WD, β_ww: 各状态间的衰减率

### 1.3 似然比检验 (LRT)

使用似然比检验判断传染效应是否显著：

- **H0**: 零假设（无传染效应，即简单泊松过程）
- **H1**: 备择假设（存在传染效应的Hawkes过程）
- 若 p值 < 0.05，则拒绝H0，认为存在显著传染效应

### 1.4 数据范围

- **分析湖泊总数**: 1,009,910 个
- **使用的分位数阈值**: 0.90 (即10%和90%分位数作为极端事件边界)

## 5. 似然比检验结果

似然比检验用于判断Hawkes过程是否显著优于简单泊松过程（检验传染效应是否存在）。

### 5.1 检验结果概览

| 检验类型 | 总收敛湖泊 | 显著湖泊数 | **显著率** | 平均p值 |
|----------|-----------|-----------|-----------|---------|
| D_to_W (旱→涝传染) | 643,962 | 16,055 | **2.49%** | 0.7678 |
| W_to_D (涝→旱传染) | 643,962 | 8,372 | **1.30%** | 0.7925 |

### 5.2 关键发现

- **D_to_W检验**: 仅2.49%的湖泊显示显著的干旱→洪涝传染效应
- **W_to_D检验**: 仅1.30%的湖泊显示显著的洪涝→干旱传染效应
- 两者的平均p值都很高（约0.77-0.79），表明大多数湖泊更符合简单的泊松过程模型

## 6. 方法对比：Hawkes vs 月距平阈值

| 指标 | 月距平阈值法 | Hawkes过程法 |
|------|-------------|-------------|
| 识别的湖泊总数 | 623,981 | 643,962 (收敛) |
| 平均事件数/湖 | 2.97 | 12.53 |
| 显著敏感湖泊 | - | 16,055 (D→W) |
| 分析方法 | 确定性阈值 | 统计推断 (LRT) |
| 捕捉特性 | 极端状态跳变 | 事件传染聚类 |

### 6.1 方法差异

1. **月距平阈值法**直接检测状态跳变，识别的是"旱涝急转"事件本身
2. **Hawkes过程法**检测的是事件之间的时间依赖性（传染效应），识别的是"旱涝急转的聚类特性"

## 7. 主要发现

1. **模型收敛率适中**: 约64%的湖泊成功拟合Hawkes过程模型

2. **传染效应罕见**: 仅约2.5%的湖泊显示显著的干旱→洪涝传染效应，说明大多数湖泊的极端事件相对独立

3. **整体稳定**: 99.96%的湖泊谱半径 < 1，表明传染过程整体稳定，不会无限增长

4. **极端事件频繁**: 平均每湖有12.5个极端事件，但事件之间的时间依赖性较弱

5. **少数敏感湖泊**: 约16,000个湖泊（约2.5%）具有显著的旱涝传染效应，这些湖泊值得重点关注

6. **爆炸性湖泊极少**: 仅226个湖泊表现出爆炸性行为（可能需要特殊管理策略）

## 8. 附录：数据库表结构

### 8.1 hawkes_results 表

```sql
-- 存储拟合的Hawkes模型参数
hawkes_results(
    hylak_id,              -- 湖泊ID
    threshold_quantile,     -- 分位数阈值
    converged,             -- 是否收敛
    log_likelihood,         -- 对数似然值
    n_events,              -- 总事件数
    n_dry_events,          -- 干旱事件数
    n_wet_events,          -- 洪涝事件数
    mu_d, mu_w,            -- 基率参数
    alpha_dd, alpha_dw,     -- 传染系数
    alpha_wd, alpha_ww,
    beta_dd, beta_dw,       -- 衰减率参数
    beta_wd, beta_ww,
    spectral_radius,        -- 谱半径
    qc_pass,               -- 质量控制是否通过
    ...
)
```

### 8.2 hawkes_lrt 表

```sql
-- 存储似然比检验结果
hawkes_lrt(
    hylak_id,
    threshold_quantile,
    test_name,              -- 'D_to_W' 或 'W_to_D'
    lr_statistic,          -- 似然比统计量
    p_value,               -- p值
    reject_null,           -- 是否拒绝零假设
    ...
)
```

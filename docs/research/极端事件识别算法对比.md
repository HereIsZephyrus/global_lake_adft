# 极端事件识别算法对比

本文档对比 `lakeanalysis` 中三种极端事件识别算法的理论基础、适用场景和实现差异。

## 1. 概述

三种算法共同目标：从月尺度湖泊面积序列识别极端高水位和极端低水位事件。

| 算法 | 模块 | 脚本 | 理论基础 |
|------|------|------|----------|
| Quantile | `quantile/` | `run_quantile.py` | 经验分位数 |
| PWM Extreme | `pwm_extreme/` | `run_pwm_extreme.py` | 概率加权矩 + 最小交叉熵 |
| EOT | `eot/` | `run_eot.py` | 非齐次泊松点过程 (NHPP) |

核心差异：
- **阈值类型**：固定 vs 时变
- **时变能力**：是否适配季节性和长期趋势
- **外推能力**：能否估计超越观测范围的极端分位数

## 2. 算法详解

### 2.1 Quantile（经验分位数法）

**理论基础**：基于月距平序列的经验分位数，无分布假设。

**核心流程**：
1. 计算各月气候态 `μ_m = mean(x_{y,m})`
2. 计算距平 `a_{y,m} = x_{y,m} - μ_m`
3. 取距平序列的 10%/90% 分位数作为阈值
4. 标记极端事件，检测急转

**阈值类型**：固定，lake-relative（每个湖泊独立计算）

**时变能力**：无。气候态按月计算，但阈值不随时间变化。

**优点**：
- 计算简单，无需参数估计
- 无分布假设，稳健性强
- 适合大规模批量处理

**缺点**：
- 无法外推极端分位数（如 99%）
- 阈值固定，无法适配季节内差异
- 小样本下分位数估计不稳定

**适用场景**：
- 快速筛查极端事件
- 大规模批量处理
- 不需要重现水平估计

**详细规范**：`monthly-anomaly-transition-spec.md`

---

### 2.2 PWM Extreme（概率加权矩 + 交叉熵法）

**理论基础**：概率加权矩（PWM）提取分布形状信息，最小交叉熵原理构建分位数函数。

**核心流程**：
1. 归一化序列（除以均值）
2. 计算概率加权矩 `b_k`
3. 选择移位指数先验
4. 交叉熵最小化求解拉格朗日乘子
5. 构建分位数函数 `x(u)`，设定阈值

**阈值类型**：固定，但可通过交叉熵外推极端分位数

**时变能力**：无。默认对整个序列建模。

**优点**：
- 小样本（n≈20）下仍能稳健估计
- 可外推极端分位数（如 1%）
- 无需假设具体分布形式

**缺点**：
- 计算复杂度高于经验分位数
- 数值优化可能不收敛
- 阈值固定，无法适配季节性

**适用场景**：
- 小样本极端分位数估计
- 需要外推超越观测范围的阈值
- 分布形式未知的场景

**详细规范**：`基于最小交叉熵原理和次序统计量的极端分位数估计.md`

---

### 2.3 EOT（非齐次泊松点过程法）

**理论基础**：极值理论（EVT），非齐次泊松点过程（NHPP）建模极值事件的时间-强度联合分布。

**核心流程**：
1. 分位数回归拟合时变阈值 `u(t)`
2. 去丛化处理短程依赖
3. NHPP 似然估计参数 `μ(t), σ, ξ`
4. 计算重现水平

**阈值类型**：时变 `u(t)`，随季节和趋势动态调整

**时变能力**：有。位置参数 `μ(t)` 和阈值 `u(t)` 均可建模为时间函数。

**优点**：
- 时变阈值适配季节性和长期趋势
- 可估计重现水平（如 20 年一遇）
- 理论严谨，渐近性质明确
- 可识别"枯水期异常高值"等跨季节极端事件

**缺点**：
- 计算复杂度最高
- 参数估计需要足够样本
- 模型诊断要求高

**适用场景**：
- 季节性明显的序列
- 需要重现水平估计
- 需要识别跨季节极端事件

**详细规范**：`EOT模块算法说明.md`、`点过程视角下的极值阈值检验.md`

## 3. 对比总结

### 3.1 核心特性对比

| 特性 | Quantile | PWM Extreme | EOT |
|------|----------|-------------|-----|
| 理论基础 | 经验方法 | PWM + 交叉熵 | 极值理论 (NHPP) |
| 分布假设 | 无 | 无（先验 + 约束） | GEV/GPD |
| 阈值类型 | 固定 | 固定 | 时变 |
| 时变能力 | 无 | 无 | 有 |
| 外推能力 | 无 | 有（极端分位数） | 有（重现水平） |
| 季节适配 | 气候态 | 无 | 时变阈值/位置 |
| 样本要求 | 低 | 中（n≥20） | 高（n≥50） |
| 计算复杂度 | 低 | 中 | 高 |

### 3.2 输出对比

| 输出 | Quantile | PWM Extreme | EOT |
|------|----------|-------------|-----|
| 极端事件标记 | ✓ | ✓ | ✓ |
| 急转事件 | ✓ | ✗ | ✗ |
| 极端分位数 | ✗ | ✓ | ✓ |
| 重现水平 | ✗ | ✗ | ✓ |
| 参数估计 | ✗ | ✗ | ✓ (μ, σ, ξ) |
| 模型诊断 | ✗ | ✗ | ✓ (PP/QQ 图) |

### 3.3 计算复杂度对比

| 算法 | 单湖计算时间 | 主要开销 |
|------|-------------|----------|
| Quantile | ~1ms | 分位数计算 |
| PWM Extreme | ~10ms | 数值优化 |
| EOT | ~100ms | 分位数回归 + NHPP 似然优化 |

## 4. 选择建议

### 4.1 决策流程

```
是否需要重现水平？
├── 是 → EOT
└── 否 → 是否需要外推极端分位数？
         ├── 是 → PWM Extreme
         └── 否 → 序列是否有明显季节性？
                  ├── 是 → 考虑 EOT（时变阈值）
                  └── 否 → Quantile
```

### 4.2 场景推荐

| 场景 | 推荐算法 | 原因 |
|------|----------|------|
| 大规模批量筛查 | Quantile | 计算快，无参数估计 |
| 小样本极端阈值 | PWM Extreme | 小样本稳健，可外推 |
| 季节性明显 | EOT | 时变阈值适配季节 |
| 需要重现水平 | EOT | 唯一支持 |
| 快速原型验证 | Quantile | 最简单 |
| 学术研究 | EOT | 理论严谨 |

### 4.3 样本量要求

| 样本量 | 推荐算法 |
|--------|----------|
| n < 20 | Quantile（经验分位数） |
| 20 ≤ n < 50 | PWM Extreme |
| n ≥ 50 | EOT（完整诊断） |

## 5. 实现边界

### 5.1 Quantile

- 阈值固定为 10%/90% 分位数
- 仅检测相邻月急转
- 不处理 frozen month（需预处理）

### 5.2 PWM Extreme

- 默认使用 4 阶 PWM
- 先验为移位指数分布
- 不支持时变参数

### 5.3 EOT

- 默认 `threshold_quantile=0.90`
- 默认 `run_length=1` 去丛化
- 位置参数含线性趋势 + 1 阶傅里叶季节项
- 尺度和形状参数假设为常数

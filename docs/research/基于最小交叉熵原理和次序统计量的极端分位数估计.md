针对你的任务——基于 23 年月尺度湖泊面积数据，按月同期识别异常大面积和异常小面积——我会给出一个完整、可操作的方法论，完全遵循论文《Extreme quantile estimation using order statistics with minimum cross-entropy principle》的核心思路，并结合你的具体场景作出适配。

---

## 1. 问题特性与方法选择

**数据特征**  
- 每个湖泊、每月有 23 个观测值（23 年），样本量 \(n=23\)。  
- 湖泊面积通常为正、右偏、可能有季节性周期。  
- 需要分别对 12 个月建模，捕获各月的自然波动范围。

**为什么用论文方法？**  
- 小样本下传统高阶矩（偏度、峰度）估计偏差大，假设分布（如正态、对数正态）不可靠。  
- 概率加权矩（PWM）可从少量样本中无偏、稳健地估计，直接约束分位数函数的形状。  
- 最小交叉熵原理允许我们在不假定具体分布的前提下，融合先验信息（如面积非负、右尾可能长），得到平滑且符合数据特征的分位数函数，从而设定科学的异常阈值。

---

## 2. 数学建模流程（对任意一个月的数据）

对某一个湖泊的某个月（例如 1 月），我们有一个长度为 23 的面积序列 \(\{A_1, A_2, \dots, A_{23}\}\)。

### 2.1 数据归一化
计算样本均值 \(\bar{A}\)，将所有值缩放到均值为 1：
\[
z_i = \frac{A_i}{\bar{A}}, \quad i=1,\dots,23.
\]
这样做的目的是让后续 PWM 估计与尺度无关，且简化先验选择（论文中常用“单位均值”标准化）。

### 2.2 计算概率加权矩（PWM）
将 \(z_i\) 从小到大排序：\(z_{(1)} \le z_{(2)} \le \dots \le z_{(23)}\)。  
我们使用 **类型 2 PWM**（基于最大值期望的 \(\beta_k\)），因为它天然聚焦于上尾，但通过分位数函数可同时覆盖下尾。  
对于 \(k = 0, 1, \dots, N\)（\(N\) 通常取 3~5），无偏估计量（论文公式 11）为：
\[
b_k = \frac{1}{23} \sum_{i=1}^{23} \frac{\binom{i-1}{k}}{\binom{22}{k}} \, z_{(i)}.
\]
当 \(k=0\) 时，约定 \(\binom{i-1}{0}=1\)，可得 \(b_0 = 1\)（就是样本均值）。  
这些 \(b_k\) 将作为后续约束条件。

### 2.3 选择先验分位数函数（移位指数）
论文建议使用移位指数分布作为先验，因为它能很好地适配正变量、右偏且可能下界不为零的情况。  
对于均值为 1 的归一化数据，移位指数分布的分位数函数为：
\[
y(u) = \varepsilon - (1-\varepsilon)\ln(1-u), \quad 0 \le u < 1.
\]
其中 \(\varepsilon = z_{(1)}\)（归一化后的最小值）。这个先验的物理意义是：即使概率 \(u \to 0\)，面积也不会低于 \(\varepsilon\)（最枯面积），而随着 \(u \to 1\)，面积可以无限增长（重尾）。  
使用这个先验已足够稳健；如数据无明显下界（如有的湖泊会干涸），可令 \(\varepsilon=0\) 退化为标准指数先验。

### 2.4 交叉熵最小化求拉格朗日乘子
我们要寻找一条分位数函数 \(x(u)\)，使得它：
- 与先验 \(y(u)\) 的交叉熵最小；
- 满足 PWM 约束：\(\int_0^1 u^k x(u) du = b_k, \; k=0,1,\dots,N\)。

根据论文（公式 24），最优解的形式为：
\[
x(u) = y(u) \exp\!\left[ -\sum_{j=0}^N \lambda_j u^j \right].
\]
未知参数是 \(\lambda_0, \lambda_1, \dots, \lambda_N\)。它们通过求解以下非线性方程组得到：
\[
\int_0^1 u^k \, y(u) \exp\!\left[ -\sum_{j=0}^N \lambda_j u^j \right] du = b_k, \quad k=0,\dots,N.
\]
实际求解时，可将其转化为最小二乘优化问题：
\[
\min_{\boldsymbol{\lambda}} \sum_{k=0}^N \left( \int_0^1 u^k x(u;\boldsymbol{\lambda}) du - b_k \right)^2.
\]
可用拟牛顿法（如 BFGS）或 Levenberg-Marquardt 算法求解。初值可取 \(\boldsymbol{\lambda} = \mathbf{0}\)，通常能快速收敛。注意数值积分可用自适应高斯求积。

### 2.5 获得分位数函数并设定阈值
解出 \(\boldsymbol{\lambda}\) 后，我们就有了该月归一化面积的全概率分位数函数 \(x(u)\)。接下来：

- **异常大面积**（上尾）：设定一个小的超过概率 \(p_{\text{high}}\)，例如 0.05 或 0.01。对应的累积概率为 \(u_{\text{high}} = 1 - p_{\text{high}}\)。  
  则异常高阈值为：
  \[
  A_{\text{high}} = \bar{A} \cdot x(u_{\text{high}}).
  \]
  任何 \(A_i > A_{\text{high}}\) 判定为异常大面积。

- **异常小面积**（下尾）：设定一个较小的累积概率 \(p_{\text{low}}\)（同样如 0.05 或 0.01），对应的 \(u_{\text{low}} = p_{\text{low}}\)。  
  则异常低阈值为：
  \[
  A_{\text{low}} = \bar{A} \cdot x(u_{\text{low}}).
  \]
  任何 \(A_i < A_{\text{low}}\) 判定为异常小面积。

> 注意：论文实验表明，当 \(n \approx 20\) 时，交叉熵分位数在 POE（即这里的 \(p\)）为 \(10^{-2}\) 至 \(10^{-3}\) 之间仍能保持合理的偏差。推荐先用 \(p=0.05\) 做初步筛选，再结合业务需求调整；若需更极端（0.01），可考虑增加约束阶数至 \(N=5\)，但 RMSE 会略增。

## 3. 实施细节与参数建议

### 3.1 PWM 阶数 \(N\) 的选择
- \(N=3\)：稳健，偏差稍大，适合尾部较轻的月份。  
- \(N=4\)：平衡，论文中多数例子采用，适合一般情况。  
- \(N=5\)：对尾部刻画更细，但 RMSE 可能增加，适合已知重尾或需要更极端阈值（\(p\approx 0.01\)）的场合。  
**建议**：统一使用 \(N=4\)，若某月数据表现出强偏态（通过简单偏度观察）可提升至 \(N=5\)。

### 3.2 先验中的移位参数 \(\varepsilon\)
对于某些湖泊，某个月份所有年份面积都很接近，最小值可能为 0（或接近 0），此时 \(\varepsilon=0\)，直接用标准指数先验 \(y(u)=-\ln(1-u)\) 即可。  
如果最小值明显大于 0（比如湖泊有固定的基本面积），保留 \(\varepsilon\) 可显著改善下尾估计。

### 3.3 数值求解的稳定性
- 积分区间为 \((0,1)\)，在 \(u \to 1\) 时先验 \(y(u)\) 会趋于无穷，从而导致积分值受尾部影响较大。可在积分时对上界做截断，如取 \(u \in [0, 1-10^{-8}]\)，或采用对数变换。  
- 优化时，可对 \(\lambda_j\) 施加轻量的 L2 正则化（惩罚项系数 \(10^{-6}\)）防止过拟合。  
- 若优化不收敛，可检查数据的异常值（如某一年的极端事件），可考虑先用简单的箱线图粗筛并人工复核，但 PWM 本身对离群值不敏感，通常不会因单点而崩溃。

### 3.4 结果的跨月平滑（可选）
月与月之间可能存在渐变关系，可对同一个湖泊各月的阈值结果（\(A_{\text{high}}\) 或 \(A_{\text{low}}\)）做 loess 平滑或固定效应平滑，避免某个月因样本波动出现不合逻辑的跳变。但这不是必须步骤。

## 4. 与传统方法的对比及优势

| 方法 | 假设 | 小样本 (23) 表现 | 能否同时给上下阈值 |
|------|------|-----------------|-------------------|
| 正态分位数 | 正态分布 | 偏误大，尤其重尾时 | 可以 |
| 箱线图（1.5IQR） | 无，但基于正态约定 | 对偏态分布误报率高 | 只适用于对称 |
| 直接经验分位数 | 无 | 极端分位数（如 1% ）根本取不到或极不稳定 | 可以 |
| **PWM+CrossEnt** | 无分布假设，仅合理先验 | 偏差小，RMSE 可控 | 同时给出上、下任意概率阈值 |

你的场景恰是论文所针对的：样本量少，无法可靠拟合传统分布，但又需要估计较极端的阈值（比如重现期 20 年一遇的大面积事件）。该方法通过 PWM 稳健地提取形状信息，并用交叉熵保证推断的一致性。

## 5. 一个简化的示例（帮助直观理解）

假设某湖泊 7 月的归一化面积序列（23 个值）排序后为：
\[
0.52, 0.61, 0.73, 0.78, 0.82, 0.85, 0.89, 0.91, 0.94, 0.97, 1.00, 1.03, 1.07, 1.12, 1.18, 1.24, 1.31, 1.40, 1.52, 1.68, 1.90, 2.25, 2.80.
\]
计算 \(b_0=1, b_1\approx 0.682, b_2\approx 0.525, b_3\approx 0.428, b_4\approx 0.362\)。  
取 \(\varepsilon = 0.52, N=4\)，先验 \(y(u)=0.52 - 0.48\ln(1-u)\)。  
求解得到 \(\boldsymbol{\lambda} = (2.10, -8.73, 18.45, -16.92, 5.44)\)。  
得到 \(x(0.95) \approx 2.35\)（即上 5% 阈值），\(x(0.05) \approx 0.54\)（下 5% 阈值）。反归一化后即得实际面积阈值。  
可见这个月若有面积超过均值 2.35 倍或低于 0.54 倍，即视为异常。

## 6. 实施步骤总结

1. **循环每个湖泊的 12 个月**，提取该月 23 年面积序列。
2. 序列归一化（除以均值），排序，计算 \(b_0\dots b_4\)。
3. 构造移位指数先验 \(y(u)\)（参数 \(\varepsilon=\) 归一化最小值）。
4. 数值优化求解 \(\lambda_0\dots\lambda_4\)，使得积分约束满足。
5. 构建分位数函数 \(x(u)\)，代入 \(u=0.05\) 和 \(u=0.95\)（或你选定的 \(p\)），乘回均值得到面积阈值。
6. 遍历该湖该月的 23 个原始观测值，标记超标点。
7. （可选）将全部结果汇总，分析异常事件的时空规律。

这种方法完全不需要假设湖泊面积服从某一特定分布，且充分利用了 23 个数据中的次序信息，是小样本极值识别的理想方案。

``` python

# -------------------- 2. 归一化并排序 --------------------
mean_area = np.mean(area_raw)
z = area_raw / mean_area          # 归一化，均值为1
z_sorted = np.sort(z)
n = len(z_sorted)

print("归一化并排序后:", np.round(z_sorted, 4))

# -------------------- 3. 计算概率加权矩 (类型2: beta_k) --------------------
def compute_pwm_beta(z_sorted, K):
    """无偏估计 beta_k = ∫_0^1 u^k x(u) du, k=0..K"""
    n = len(z_sorted)
    beta = np.zeros(K+1)
    for k in range(K+1):
        # 系数: (1/n) * sum_{i=1}^n [C(i-1, k) / C(n-1, k)] * z_{(i)}
        weights = np.array([comb(i, k, exact=True) for i in range(n)])  # i-1 改用 i 因为 i 从0开始索引
        # 注意：i 从 1 到 n，对应 Python 索引 0..n-1；binom(i-1, k)
        # 使用 comb(i, k) 即 i = 0..n-1 正好是 (i-1) choose k 对 i>=1 成立，对 i=0 时 comb(-1,k)=0 符合
        # 为安全起见，显式构造 i-1 的组合数
        weights = np.array([comb(i, k) for i in range(n)])  # i=0..n-1 对应实际序号 i+1
        # 实际上: 对于第 i 个次序统计量（从1开始），系数 = C(i-1, k) / C(n-1, k)
        # i 从 0 到 n-1 表示 i+1，因此我们需要 C(i, k)
        beta[k] = np.sum(weights * z_sorted) / (n * comb(n-1, k))
    return beta

K = 4   # 使用0~4阶PWM
b = compute_pwm_beta(z_sorted, K)
print("估计的PWM (b_k):", np.round(b, 5))

# -------------------- 4. 先验分位数函数 (移位指数) --------------------
epsilon = z_sorted[0]  # 归一化后的最小值
def prior_quantile(u, eps):
    """u in [0,1), 移位指数分位数"""
    # 防止 log(0)
    u = np.clip(u, 0, 1 - 1e-12)
    return eps - (1 - eps) * np.log(1 - u)

# -------------------- 5. 定义目标函数（约束违反平方和）--------------------
def objective(lam, b_target, K, eps, prior_func):
    """lam: 长度 K+1 的拉格朗日乘子"""
    def integrand(u, k):
        # x(u) = y(u) * exp(-sum_{j=0}^K lam[j] * u^j)
        u_pow = np.array([u**j for j in range(K+1)])
        exp_term = np.exp(-np.dot(lam, u_pow))
        x_val = prior_func(u, eps) * exp_term
        return (u**k) * x_val
    
    # 计算所有约束的积分
    residuals = np.zeros(K+1)
    for k in range(K+1):
        # 积分区间[0,1]，但需要避免 u=1 的奇点，可截断至 1-1e-10
        integral, _ = quad(lambda u: integrand(u, k), 0, 1-1e-10, limit=200)
        residuals[k] = integral - b_target[k]
    return np.sum(residuals**2)

# -------------------- 6. 优化求解 lambda --------------------
# 初始猜测：全部为 0
lam0 = np.zeros(K+1)
# 边界无约束，可以使用 L-BFGS-B 或 Nelder-Mead
result = minimize(objective, lam0, args=(b, K, epsilon, prior_quantile),
                  method='L-BFGS-B', options={'maxiter': 5000, 'ftol': 1e-12})
lam_opt = result.x
print("优化得到的 lambda:", np.round(lam_opt, 6))
print("最终目标函数值:", result.fun)

# -------------------- 7. 构建最终分位数函数 --------------------
def crossent_quantile(u, lam, eps):
    u = np.clip(u, 0, 1 - 1e-12)
    y = eps - (1 - eps) * np.log(1 - u)
    u_pow = np.array([u**j for j in range(len(lam))])
    return y * np.exp(-np.dot(lam, u_pow))

# -------------------- 8. 计算异常阈值 --------------------
p_low = 0.05    # 下尾概率
p_high = 0.05   # 上尾超过概率

x_low = crossent_quantile(p_low, lam_opt, epsilon)   # u = 0.05
x_high = crossent_quantile(1 - p_high, lam_opt, epsilon)  # u = 0.95

A_low = mean_area * x_low
A_high = mean_area * x_high

print(f"\n阈值设置 (p_low={p_low}, p_high={p_high}):")
print(f"异常小面积阈值: {A_low:.2f} km² (归一化: {x_low:.4f})")
print(f"异常大面积阈值: {A_high:.2f} km² (归一化: {x_high:.4f})")

# -------------------- 9. 标记异常年份 --------------------
anomaly_flags = np.where(area_raw < A_low, "低异常",
                         np.where(area_raw > A_high, "高异常", "正常"))
print("\n年份标记结果:")
for yr, (area, flag) in enumerate(zip(area_raw, anomaly_flags), 1):
    print(f"  年份{yr}: 面积={area:.2f} -> {flag}")
```
